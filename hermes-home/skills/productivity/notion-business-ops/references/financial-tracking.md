# Финансовый учёт: полный workflow

## 1. Создание базы данных

```python
mcp_mcpmarket_notion_notion_create_database(
    title="Финансы <Название>",
    description="Учёт расходов и доходов",
    schema='''
CREATE TABLE (
  "Название" TITLE,
  "Тип" SELECT('Расход':red, 'Доход':green),
  "Сумма" NUMBER,
  "Категория" SELECT(
    'Материалы':brown, 'Инструмент':gray, 'Транспорт':yellow,
    'Реклама':blue, 'Связь':orange, 'Аренда':purple,
    'Личное':pink, 'Прочее':gray,
    'Установка замка':green, 'Вскрытие':red, 'СКУД':blue,
    'Видеонаблюдение':purple, 'Домофон':yellow
  ),
  "Дата" DATE,
  "Комментарий" RICH_TEXT
)'''
)
```

Результат: URL базы и `data_source_id` (collection://...).

## 2. Создание view

```python
notion_create_view(
    data_source_id="...",
    database_id="...",
    name="📊 Все операции",
    type="table"
)
```

## 3. Добавление записи

```python
notion_create_pages(
    parent={"data_source_id": "..."},
    pages=[{
        "properties": {
            "Название": "Краткое описание",
            "Тип": "Расход",           # или "Доход"
            "Сумма": 3000,              # число, не строка
            "Категория": "Материалы",
            "date:Дата:start": "2026-06-10",
            "date:Дата:is_datetime": 0,
            "Комментарий": "Детали покупки/заказа"
        }
    }]
)
```

**Важно:** сумма — число (int/float), не строка. Дата — развёрнутый формат.

## 4. Добавление новой категории

```python
notion_update_data_source(
    data_source_id="...",
    statements='ALTER COLUMN "Категория" SELECT('
        '"Материалы":brown, "Инструмент":gray, ..., "Новая":pink'
    ')'
)
```

Нужно передать **полный** список опций, не только новую.

## 5. Cron для ежемесячной аналитики

```python
cronjob(
    action="create",
    name="Финансовая сводка",
    schedule="0 9 1 * *",          # 1-го числа в 9:00
    skills=["productivity/notion"],
    enabled_toolsets=["web", "terminal", "file", "skills"],
    prompt="""Сделай сводку по финансам за прошедший месяц:

1. Получи data_source_id = ... из базы "Финансы ..."
2. Используй notion_fetch для получения всех записей
3. Отфильтруй записи за прошлый месяц (с 1-го по последнее число)
4. Подведи итоги:
   - Общий доход (сумма строк с Тип="Доход")
   - Общий расход (сумма строк с Тип="Расход")
   - Чистая прибыль (доход - расход)
   - Топ-3 категорий расходов
   - Топ-3 категорий доходов
   - Количество заказов (строк с Тип="Доход")
5. Отправь результат пользователю в Telegram на русском языке"""
)
```
