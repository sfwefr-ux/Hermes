# Yandex Maps — Анализ конкурентов

Браузерный анализ конкурентов на Яндекс Картах: рейтинги, цены, отзывы, позиционирование. Для бизнесов в РФ и СНГ, представленных на Яндекс Картах / Яндекс Бизнес.

## Когда применять

- Анализ конкурентной среды локального бизнеса (услуги, магазины, общепит)
- Оценка позиционирования своей организации относительно конкурентов
- Поиск идей для улучшения карточки Яндекс Бизнеса
- Сравнение цен, рейтингов, количества отзывов

## Инструменты

Только `browser_navigate` + `browser_snapshot` + `browser_type` + `browser_click`. Яндекс Карты отдают контент динамически — curl/web_fetch не работают (рендерятся на клиенте). Использовать execute_code с urllib для парсинга HTML страниц карт бесполезно — нужен браузер.

## Пошаговый алгоритм

### Шаг 1: Открыть карточку своей организации

```
browser_navigate: https://yandex.ru/maps/org/{org_name}/{org_id}?si={si}
```

Со снапшота собрать:
- Рейтинг, количество оценок
- Адрес, телефон, сайт, соцсети
- График работы
- Список услуг и цен
- Особенности (способы оплаты, акции, доступность)
- Количество фото и отзывов

### Шаг 2: Найти категорию

В хлебных крошках (breadcrumbs) карточки организации кликнуть на категорию — например, «Мастерские по замкам». Яндекс покажет всех конкурентов в этой же категории на видимой области карты.

### Шаг 3: Расширить поиск

Если конкурентов мало (1-2), ввести более общий поисковый запрос в поле поиска:

```
browser_type → поле поиска → «ремонт замков установка спб»
browser_click → кнопка «Найти»
```

### Шаг 4: Собрать конкурентов

Из снапшота вытащить для каждого конкурента:
- Название и ссылку
- Рейтинг и количество оценок
- Адрес
- График работы
- Минимальную цену (если указана)
- Наличие акций/промо
- Наличие сайта

### Шаг 5: Сравнительный анализ

Свести в таблицу и выделить:
- 🥇 Лидер по рейтингу и отзывам
- 💰 Самые низкие/высокие цены
- 🏷️ Наличие акций у конкурентов
- 🌐 Наличие сайта и соцсетей
- 📍 Географическое покрытие

### Шаг 6: Рекомендации

На основе анализа дать конкретные рекомендации:
- Обновить карточку Яндекс Бизнеса (если «Давно не обновлялось»)
- Добавить услуги с ценами
- Запустить акцию (если у конкурентов есть, а у вас нет)
- Наращивать отзывы (если отрыв от конкурента небольшой)
- Указать уникальные преимущества

## Что можно извлечь из карточки

| Элемент | Где искать в снапшоте | Ценность |
|---------|----------------------|----------|
| Рейтинг | `heading + StaticText "Рейтинг" + StaticText "X,X"` | Прямое сравнение |
| Оценки | `StaticText "NN оценок"` | Объём социального доказательства |
| Цены | `button "Услуга NNN ₽"` | Ценовое позиционирование |
| Акции | `StaticText "Акция"` или `StaticText "Акции:..."` | Маркетинговая активность |
| Сайт | `link "domain.ru"` | Уровень digital-присутствия |
| Соцсети | `button "Соцсети, ..."` | Каналы коммуникации |
| Услуги | Вкладка «Товары и услуги» | Ассортимент |
| «Давно не обновлялось» | `StaticText "Давно не обновлялось"` | Сигнал к обновлению карточки |

## Редактирование карточки Яндекс Бизнеса

При попытке редактировать карточку через `browser_navigate → https://business.yandex.ru` — **всегда капча SmartCaptcha**. Пройти её через browser-инструменты невозможно.

**Обход:** пользователь заходит в Яндекс Бизнес вручную со своего устройства, я диктую что именно менять. Альтернатива — Яндекс Бизнес API с OAuth-токеном (требует настройки).

## Важные нюансы

- **Яндекс Карты режут ботов.** Работать ТОЛЬКО через browser_navigate/snapshot. Никаких curl/urllib — мгновенный 429 или пустая страница.
- **Результаты зависят от зума.** При масштабировании карты набор конкурентов меняется. Начинать с дефолтного зума карточки, потом отдалять/приближать по необходимости.
- **Фильтр «Открыто» включён по умолчанию.** Может скрывать организации, которые не отметили часы работы. При нулевых результатах — снять фильтр.
- **Модерация Яндекса — до 24 часов.** После изменения услуг, описания или акций карточка не сразу появляется в поиске. Не паниковать раньше чем через сутки. Если через 24 часа изменений нет — проверить категорию карточки в Яндекс Бизнесе.
