# BadLock — бизнес-операции

Семейный бизнес: Станислав + Надежда. Замки и системы доступа в СПб и Ленобласти.
Услуги: установка/замена/вскрытие замков, СКУД, электронные замки, домофоны, видеодомофоны, видеонаблюдение, магнитные замки на калитки, ПВЗ Wildberries.
Режим: 9:00–23:00. Оплата: карта/наличные. Гарантия: 1 год. Мин. заказ: 15 000 ₽.

## Работа с клиентами

- Представляться: «Здравствуйте! Мастер Станислав»
- Запросить фото/видео двери для оценки
- Использовать базу возражений из Базы знаний (12 тем, адаптировать под замки/СКУД)
- После монтажа: запросить отзыв на Avito и Яндекс

## Добавление записей

### Финансы
Формат от пользователя: «Расход/Заработал СУММА, КАТЕГОРИЯ/ОПИСАНИЕ»
→ Создать запись в базе «Финансы BadLock» через `notion_create_pages` с parent data_source_id.

### Монтажи
Формат: «Запиши монтаж НАЗВАНИЕ_РАБОТ на ДАТА от КЛИЕНТ»
→ Создать запись в «График монтажей». Статус по умолчанию «Подтверждён».

## Ежедневная сводка монтажей (cron)

Cron-задача: каждый вечер запрашивать «График монтажей» на завтра и отправлять сводку.
Полный workflow и настройка cron: `devops/hermes-mcp-setup` → `references/cron-notion-reminder-pattern.md`.

**Data source reference для cron-промптов:** `collection://1bd3fa40-27f0-4fc7-96b0-f7d6083ce7f0`

**Фильтр:** Статус = «Подтверждён» ИЛИ «Ожидает», Дата = завтрашняя (вычисляется агентом).

**Формат вывода:**
```
🔔 **Монтажи на завтра — {дата}**

• **{время}** — {адрес}, {имя}, {тип работ}, тел: {телефон}
```
Если заявок нет: «На завтра пока чисто. 🏠»

**Поля для вывода:** Время, Адрес, Имя клиента, Тип работ, Телефон.
